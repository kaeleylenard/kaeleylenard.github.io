Source File Revamp

In a growing design team of 8, file management gets tricky, especially when it comes to reference docs. Mercari's UX design team shares a Source File, which encapsulates handoff screens as a holistic view of the product. As more features are shipped, the Source File becomes ineffective with its outdated content, missing screens, density, and lack of contribution standards. Inspired for a revival, I spearheaded an initiative to revamp the Source File with tools ranging from a UserZoom card sort dendrogram and Figma autolayout.
